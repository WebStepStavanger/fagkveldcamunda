# fagkveldcamunda
Fra fagkveld Egil holdt 6/2-2018

#Oppsett

##Camunda Server
Last ned og installer Camunda fra https://camunda.com/download/

Kjør start-camunda.bat (eller .sh) fra katalogen du pakker opp og vipps så har du Camunda kjørende med en m2-database.

##Applikasjon med prosess
Kjør mvn install fra katalogen, kopier process-application-0.0.1-SNAPSHOT.war inn i camund-dir/server/apache-tomcat-<version>/webapps

Åpne Camunda på localhost:8080
